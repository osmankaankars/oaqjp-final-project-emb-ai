# Emotion Detector

AI-based web application that detects emotions from text using IBM Watson NLP and Flask.

## Features
- Emotion detection (anger, disgust, fear, joy, sadness)
- Dominant emotion identification
- Web interface with Flask
- Error handling for invalid input
- Unit tested and PEP8 compliant

## Technologies
- Python
- Flask
- IBM Watson NLP
- unittest
- pylint

## How to Run
```bash
python3 server.py
